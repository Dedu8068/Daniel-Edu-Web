# Daniel Ikechukwu Edu — Website

Personal portfolio website for **Daniel Ikechukwu Edu**  
(Actor • Model • Voice Artist).

Built with **Next.js (App Router)** and **Tailwind CSS**.  
Deployed on **Vercel**.

---

## Pages

- Home  
- Portfolio (Headshots / Editorial / Commercial / Digitals / Print / Stills)  
- Reels (Acting Reel, Monologues, Scenes, Vertical Reels, Clips)  
- Voice Over (Audio player with featured VO reel)  
- UGC (Vertical-first content with social links)  
- Contact (Formspree contact form, representation info, casting profiles)
